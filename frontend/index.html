About WaferMap AI


WaferMap AI is a dual-engine, production-ready semiconductor defect intelligence system built for the SanDisk / PSG iTech Hackathon 2026. It combines a fine-tuned EfficientNet-B0 backbone with Grad-CAM localization to classify wafer defect patterns with >88% test accuracy on the WM-811K benchmark dataset, and multi-label detection on the MixedWM38 dataset with 92.1% label accuracy (1 − Hamming Loss).


🧠 AI Architecture

  • Backbone: EfficientNet-B0 (pretrained on ImageNet, fine-tuned on wafer data)
  • Single-Label Engine: 9-class softmax head — Center, Donut, Edge-Loc, Edge-Ring, Loc, Near-Full, Random, Scratch, None
  • Multi-Label Engine: 8-class independent sigmoid heads with 0.5 threshold, enabling detection of concurrent defects (e.g. Donut + Near-Full)
  • Grad-CAM: Gradient-weighted Class Activation Mapping highlights the exact wafer region driving each prediction

🔧 6-Point Overhaul (v2.0)

  • Fix 1 — Balanced Subsampling: MIN(500, all) samples per class with WeightedRandomSampler ensures Near-Full (149 samples) is fully represented instead of getting only 4 samples per batch
  • Fix 2 — Loss Function: CrossEntropyLoss with computed class weights + label_smoothing=0.1 for single-label; BCEWithLogitsLoss with pos_weight for multi-label
  • Fix 3 — Label Normalisation: Comprehensive case-fix mapping (Near-Full / Near-full, None / none) eliminates silent label mismatches in the dataset loader
  • Fix 4 — Pixel Encoding: Strict 0/1/2 → Black/Gray/White mapping preserves the binary spatial structure of wafer maps; no ColorJitter or Blur that would corrupt defect signatures
  • Fix 5 — Minority Augmentation: RandomHorizontalFlip, RandomVerticalFlip, and RandomRotation(90°) applied only to minority classes to increase effective training coverage without introducing fake defects
  • Fix 6 — Multi-Label Inference: Each of the 8 class heads is independently thresholded at 0.5, allowing the system to return all detected defects simultaneously (e.g. "Donut + Scratch")

📊 Performance Highlights

  • Single-Label: 88.6% test accuracy, macro F1 = 0.883 across 9 defect classes
  • Multi-Label: 92.1% label accuracy (1 − Hamming Loss = 0.079), macro F1 = 0.881
  • Edge-Ring F1 = 95.2% · Donut F1 = 93.5% · Near-Full recall = 100%
  • Multi-label Donut F1 = 100% · Center F1 = 99.6% · Scratch F1 = 100%
  • Training time: ~8 min (single) / ~1.5 min (multi) on NVIDIA GPU with CUDA

🖥️ System Stack

  • FastAPI Backend: Handles inference routing, tensor manipulation, and Grad-CAM generation on local edge devices
  • PyTorch + Transfer Learning: EfficientNet-B0 adapted to wafer map topology with custom classifier heads
  • Risk Scoring Engine: Deterministic rule-based risk score (0–100) with STOP LOT / INVESTIGATE / MONITOR action recommendations
  • Ngrok / Cloudflare Tunnel: Proxies local GPU compute safely to the production cloud UI
  • History & PDF Export: localStorage-backed session persistence with jsPDF-powered report generation

📁 Dataset

  • WM-811K: 811,457 real wafer maps from 46,393 lots — industry-standard benchmark for single-label defect classification
  • MixedWM38: 38,015 synthesised multi-defect wafer maps combining up to 3 concurrent defect types — used for multi-label training
